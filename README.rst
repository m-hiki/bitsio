Installing
------------

Use pip to install BitsIO:

.. code-block:: bash

    pip install bitsio

Usage
------------


.. code-block:: python
   from bitsio import BitsIO

   

Example test running:

.. code-block:: bash

    python -m unittest discover ./tests/
